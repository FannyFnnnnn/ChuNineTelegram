
# 最簡單保證可以動的 Telegram 機器人

✅ **絕對可以在 Windows + Python 3.13 上面正常運作**

## 安裝
```cmd
pip install -r requirements.txt
```

## 使用步驟
1. 複製 `config.example.json` 為 `config.json`
2. 填入你的真實設定
3. 執行:
```cmd
python simple_bot.py
```

## 特點
* ✅ 只有一個外部依賴 (requests)
* ✅ 沒有任何非同步，沒有任何魔法
* ✅ 不會有任何奇怪的函式庫版本問題
* ✅ 可以接收所有類型的訊息
* ✅ 可以複製/轉發訊息
* ✅ 可以刪除訊息
* ✅ 可以回覆訊息
* ✅ 完全符合 Telegram API 標準

## 管理員指令
* `/start` - 檢查機器人是否在線
* `/ping` - 測試延遲
* `/status` - 顯示狀態

沒有任何額外的東西，就是純粹可以動的機器人。
